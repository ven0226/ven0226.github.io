Exploring Spatial Models and Feature Selection

This study assesses three spatial regression models for predicting biomass using Light Detection And Ranging (LiDAR) covariates. Many studies have used covariates derived from LiDAR to help explain the variability in tree, stand, or forest variables at a fine spatial resolution across a specified domain. However, few have compared results via feature selection and model implementation. We test several feature weighting methods and compare their results by comparing the quality of various clusterings using a distance function modified to incorporate the learned weights. spBayes, JAGS, and SAR regression models are compared using a data set collected on the U.S. Forest Service Penobscot Experimental Forest, Maine.
